How to Use Image Resizer

Using the Image Resizer tool on imagetoolify is simple and fast. Follow the steps below to resize images for PAN Card, Voter ID and other documents.

  1. Click on the Upload Image button and select your image.
  2. Select the required image size such as 20KB, 30KB, 50KB, 100KB or 300KB.
  3. If needed, enable Resize by Pixels and enter width and height.
  4. Choose the download format as Image or PDF.
  5. Click on Start Processing to resize the image.
  6. Preview the resized image and click Download.
